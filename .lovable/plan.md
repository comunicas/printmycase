

## Adicionar secret CHECKOUT_SUCCESS_TOKEN_SECRET

Usar a ferramenta `add_secret` para criar a secret `CHECKOUT_SUCCESS_TOKEN_SECRET` com um UUID v4 gerado aleatoriamente (ex: `a3f8b2c1-9d4e-4f7a-b5c6-8e2d1f0a3b7c`). Isso resolve o erro `CHECKOUT_SUCCESS_TOKEN_SECRET not configured` no checkout de cases.

Nenhuma alteração de código necessária.
